# Agriculture Region Clustering

K-Means clustering based analytical system to group agricultural regions based on crop yield characteristics and environmental factors.

## Features
- 20 region-level features including rainfall, temperature, soil properties
- Automated EDA and data preprocessing
- Feature selection using forward/backward elimination
- Optimal K determination using Elbow Method
- K-Means clustering with silhouette evaluation
- Prediction capability for new regions
